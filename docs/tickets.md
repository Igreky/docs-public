# Włączanie ticketów

!!! info "Cooldowny"
    Defaultowy cooldown ticketów: **1 minuta**

!!! example "Przykłady"
    `;ticket enable`

## Setup 

!!! example "Przykłady"
    `;ticket setup`

!!! danger "Uwaga"
    Ta komenda wyśle od razu gotowy embed z ticketem. Musisz mieć ustawioną rolę moderatora, aby tickety w pełni działały.
### Ręczne tworzenie ticketa

!!! example "Przykłady"
    `;ticket create [Treść]`
## Wyłączanie ticketów

!!! example "Przykłady"
    `;ticket disable`
