# Automoderacja

!!! tip 

    **userid** = ID użytkownika
    <p> </p>
    **user** = Użytkownik
    <p> </p>
    **reason** = Powód
    <p> </p>
    **mention** = Oznaczenie
### Banowanie
#### Użycie komendy


!!! example "Przykłady"

     `;ban [user] <reason>`
     <p> </p>
    `;ban @Korrumz2 duch`

### Odbanowywanie

!!! example "Przykłady"

     `;unban [userid]`
     <p> </p>
    `;unban 817883855310684180`

### Ostrzeganie

!!! example "Przykłady"

    `;warn [userid, mention]`
    <p> </p>
    `;unwarn [userid, mention]`
    <p> </p>
    `;warnlist [userid, mention]`
