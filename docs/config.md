# Ustawienia
!!! info "Tłumacz google"
    **variable** = Zmienna

    **value** = Wartość

## Konfiguracja kanałów

!!! example "Przykłady"

       - ;set [variable] [value]

       - ;set broadcastChannel #ogłoszenia

## Konfiguracja ról

!!! example "Przykłady"

       - ;set [variable] [value]
       - ;set mutedRole @Wyciszony 

## Konfiguracja logów

!!! example "Przykłady"
       - ;logs [variable] [value]
       - ;logs inviteCreate #kanał

!!! tip 
    Zamiast mentionować można zawsze wpisać ID. Uwaga! To nie może być kategoria albo kanał głosowy
